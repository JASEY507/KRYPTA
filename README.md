# KRYPTA
KRYPTA // DATA FORTRESS CORE  AES-256-CBC ve PBKDF2 yardımıyla yerel verilerin gizliliğini koruyan, çok iş parçacıklı siber güvenlik modülüdür.  Her dosya için benzersiz SHA-256 bütünlük imzasını yerel veri tabanında doğrular; manipülasyonları engeller. DoD standartlarında güvenli silme (shred) ve anlık terminal log akışı içerir.
