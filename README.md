# KRYPTA
KRYPTA // DATA FORTRESS CORE  AES-256-CBC ve PBKDF2 yardımıyla yerel verilerin gizliliğini koruyan, çok iş parçacıklı siber güvenlik modülüdür.  Her dosya için benzersiz SHA-256 bütünlük imzasını yerel veri tabanında doğrular; manipülasyonları engeller. DoD standartlarında güvenli silme (shred) ve anlık terminal log akışı içerir.



Windows Command Prompt / PowerShell
`python -m pip install --upgrade pip && pip install -r requirements.txt && python main.py`




Linux Ubuntu / Debian / Kali Linux
`sudo apt update && sudo apt install -y python3-pip python3-tk && python3 -m pip install --upgrade pip && pip3 install -r requirements.txt && python3 main.py`




macOS Terminal
`python3 -m pip install --upgrade pip && pip3 install -r requirements.txt && python3 main.py`
